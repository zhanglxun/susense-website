核心模块

四大模块，覆盖专注力训练全生命周期

桌面客户端训练系统

解谜、记忆、逻辑推理、数学挑战等 57 款互动认知训练任务，基于 Phaser 引擎开发，流畅稳定的游戏化体验。系统自动追踪训练进度，根据表现动态调整难度，数据采集持续优化个人训练方案。

  • Electron 跨平台（Win / Mac / Linux）一套代码三端覆盖
  • Phaser 引擎，支持物理引擎与模块化扩展
  • 离线可用 + 在线数据同步双模式
移动端协同

与桌面端训练进度实时同步，随时随地继续训练。智能提醒与通知帮助儿童建立稳定的训练习惯，家长端随时查看训练报告与数据分析。

  • 打通居家、通勤、课余的碎片化时间
  • 显著提升训练依从性与坚持率
  • 家长端可视化训练报告与进度追踪
机构管理后台

面向机构管理者的企业级运营系统，支持多门店 / 多区域统一管理与分权运营。让机构真正掌握每一个学员的训练数据，用数据驱动机构运营与效果证明。

  • 多维度用户管理：账户、权限、隐私、分组
  • 训练计划配置与个性化方案管理
  • Spring Security 精细权限控制，保障数据安全
脑波硬件集成

脑电生物反馈设备 SDK 驱动已完成对接调通，Demo 演示界面已实现脑波数值实时采集与动态展示，支持脑波数据与训练系统数据联动分析。

  • 将不可见的"注意力"变成可量化的客观数据
