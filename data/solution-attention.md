## 第二区块 | 行业痛点 × 市场机遇

### 区块标题
```
一个被严重低估的市场，正在等待系统性解决方案
```

### 三大痛点卡片

**痛点一：需求巨大，但解决方案极度分散**
> 中国ADHD儿童确诊人数达2300万，而市面上的解决方案要么是高门槛药物治疗，要么是缺乏科学依据的训练App，家长找不到可信赖的系统性干预方案。

**痛点二：线下机构有经验，但无法规模化**
> 优质的专注力训练机构积累了大量一线经验和案例数据，但依赖真人导师、难以标准化复制，服务能力被严重束缚在物理空间内。

**痛点三：软硬件碎片化，数据无法形成闭环**
> 评测工具、训练内容、脑波设备、家长管理系统各自独立，数据无法打通，训练效果难以量化追踪，机构管理成本极高。

### 市场机遇说明
```
数字疗法（DTx）正在重塑神经/精神类疾病的干预方式。
软件作为治疗手段，已获得 FDA 及 NMPA 的认可与关注。
谁能率先在中国市场建立起"评估-训练-支持"的数据闭环，谁就能构建不可逾越的先发壁垒。
```

## 第四区块 | 四大核心模块详解

```
四大模块，覆盖专注力训练全生命周期
```

### 模块一：桌面客户端训练系统

**核心功能**
- 解谜游戏、记忆游戏、逻辑推理、数学挑战、大脑训练等九大维度 21 类互动任务
- 基于 Phaser 引擎开发，流畅稳定的游戏化训练体验
- 系统自动追踪用户训练进度，根据表现动态调整难度
- 数据采集与分析，持续优化个人训练方案

**技术亮点**
- 基于 **Electron 跨平台框架**，Windows / Mac / Linux 一套代码三端覆盖
- 内置 **Phaser 游戏引擎**，支持物理引擎、动画系统、模块化扩展
- 离线可用 + 在线数据同步双模式

**设计备注**
> 配合界面截图/录屏 GIF，重点突出游戏画面的趣味性和儿童友好感

---

### 模块二：移动端

**核心功能**
- 与桌面端训练进度实时同步，随时随地继续训练
- 智能提醒与通知，帮助儿童建立稳定的训练习惯
- 家长端查看训练报告与数据分析

**核心价值**
> 让专注力训练不再局限于固定设备，打通居家、通勤、课余的碎片化时间，显著提升训练依从性。

---

### 模块三：管理后台

**核心功能**
- 多维度用户管理：账户、权限、隐私、分组
- 训练计划配置：创建、分配、调整个性化方案
- 数据分析与可视化报告：训练频次、能力变化趋势、效果对比
- 多门店 / 多区域支持：统一管理，分权运营
- 系统安全框架：基于 Spring Security，权限精细控制

**核心价值**
> 让机构管理者真正掌握每一个学员的训练数据，摆脱"凭感觉"做决策，用数据驱动机构运营与效果证明。

---

### 模块四：脑波硬件集成

**现状**
- 脑电生物反馈设备 SDK 驱动已完成对接调通
- Demo 演示界面已实现脑波数值实时采集与动态展示
- 支持脑波数据与训练系统数据联动分析

**核心价值**
> 将不可见的"注意力"变成可量化的客观数据，为训练效果提供最有说服力的科学支撑，同时为未来 AI 多模态评估奠定数据基础。

**设计备注**
> 可配合脑波采集设备实物图、实时数值波形动态图展示

## 第五区块 | 技术架构与优势

### 区块标题
```
企业级技术底座，支撑从单点验证到规模化运营的每一步
```

### 技术选型一览表

| 层级 | 技术方案 | 选型理由 |
|------|----------|----------|
| **客户端** | Electron | 跨平台覆盖，Web技术开发，自动更新支持 |
| **游戏引擎** | Phaser | 轻量高性能，支持物理引擎与插件扩展 |
| **后端框架** | Java / SpringBoot | 企业级标准，支持高并发与高可用 |
| **数据库** | MySQL | 成熟稳定，支持复杂查询与数据分析 |
| **安全框架** | Spring Security | 精细权限控制，保障用户数据安全 |
| **Web服务** | Nginx | 高性能负载均衡，前端静态资源分发 |
| **服务器** | Linux（腾讯云） | 弹性扩容，持续集成部署 |

### 五大技术优势

**① 跨平台，快速覆盖多端**
一套客户端代码，同步支持 Windows / Mac / Linux，大幅降低多端维护成本。

**② 高可用后端，支持业务规模化**
Java 企业级架构设计，支持未来高并发、大数据量场景，不因用户增长而重构。

**③ 游戏内容生态可持续扩展**
基于 Phaser 引擎的模块化设计，新游戏内容可快速接入，不影响现有系统稳定性。

**④ 多门店多区域统一管理**
后台架构从设计之初即支持多租户、多层级权限模型，满足连锁机构的管理诉求。

**⑤ 软硬件闭环，数据互通**
脑波硬件数据与游戏训练数据打通，为后续 AI 自适应训练引擎提供完整数据基础。
