# M6 — 中银国际案例页 case-bank.html 新建计划

## 叙事结构（故事型）

### 1. 项目背景
- 客户：中银国际证券（BOCI Securities）
- 行业：金融证券
- 项目：移动端 App 全面升级重构
- 时间：2025年

### 2. 核心挑战
- 原有 App 界面老旧，交互复杂，新用户上手门槛高
- 交易流程冗长，关键操作步骤过多，用户流失率高
- 功能堆砌但缺乏体验设计，与竞品差距明显

### 3. 我们的方案
- 深入用户研究：访谈目标用户，分析交易行为路径
- 全面重设计：信息架构重组 + 视觉语言升级
- 关键流程优化：交易下单、资产查看、行情浏览等核心链路简化
- 新增功能：智能投顾入口、资产配置模块、个性化行情推送

### 4. 交付成果
- UI 设计稿（关键页面截图占位）
- 原型与交互规范
- 设计系统（组件库）
