## 核心能力

### 1. Claude Code 企业培训
图标建议：`book-open`（翻开的书）

Claude Code 是目前最强的 AI 编程助手之一，但真正让团队用起来，需要的不是介绍 PPT，而是实战陪跑。

我们提供：
- **企业内训（上门）**：结合你们的真实项目和代码库，手把手拆解如何用 Claude Code 提速
- **工作坊 / 实战营**：小班制，边学边做，每人完成一个真实任务
- **线上直播课**：灵活安排，适合分布式团队
- **长期陪跑顾问**：不只培训，持续辅导团队在真实项目中落地

不讲理论，直接上手。培训结束，团队成员能独立用 AI 完成日常开发任务。

> 同类工具如 OpenClaw 也可纳入培训体系，帮助团队配置个人 AI 工作助理。

### 2. Agent 定制开发
图标建议：`cpu`（芯片）或 `bot`

为你的业务打造专属 AI 员工，替代重复、低价值的人工操作。

**我们擅长的 Agent 类型：**

| Agent 类型 | 典型场景 |
|-----------|---------|
| 工作流自动化 Agent | 自动处理报表、邮件回复、表单填报、数据整理 |
| 知识问答 Agent | 基于企业文档的智能客服、内部知识助手 |
| 代码辅助 Agent | 帮开发团队审查代码、生成测试、写文档 |
| 业务决策辅助 Agent | 分析销售数据、生成经营建议、辅助判断 |

Agent 不是通用工具，而是针对你的业务流程和数据深度定制的。

### 3. RAG 知识库构建
图标建议：`database`（数据库）

让 AI 读懂你们自己的文档、制度、产品手册、历史数据——而不是只会回答通用问题。

- 支持 PDF、Word、Excel、网页、数据库等多种数据源接入
- 构建私有知识库，AI 回答有据可查，可溯源
- 适用于客服机器人、内部助手、合规审查等场景

### 4. 私有化部署
图标建议：`shield`（盾牌）或 `server`

数据不出企业，合规放心用。

- 在企业本地或私有云环境部署 AI 能力
- 支持与现有系统（OA、ERP、CRM）集成
- 适合有数据安全要求的医疗、金融、政府类客户

## 我们的服务方式

**不卖课，卖结果。**

很多 AI 培训讲完就走，团队回去还是不会用。我们的目标是：服务结束后，你的团队能独立推进 AI 落地，而不是依赖我们。

```
需求诊断 → 方案设计 → 实施/培训 → 落地验收 → 持续顾问（可选）
```

- 先搞清楚你们现在卡在哪里，再决定做什么
- 培训、开发、部署可以组合，也可以单独来
- 支持按天、按项目、按月多种合作模式
