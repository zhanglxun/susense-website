# 速睿森视官网 — 变更历史

---

## 2026年3月 — 官网重构

### 背景
将2020年代外包公司官网，升级为2025年以AI应用+咨询为主的专业服务公司形象。

### 主要变更

#### 1. 导航结构重构
**旧结构**：
- 方案与服务（双列下拉：解决方案+咨询服务）
- 产品系列（数智织机、TempoSnap、TempoFlow）
- 客户案例

**新结构**：
- 服务与方案（左右两栏卡片式下拉）
  - 服务：软件系统开发、产品体验设计、AI赋能服务
  - 方案：医疗健康、智能制造、注意力培训
- 产品系列（仅保留数智织机）
- 客户案例
- 关于我们

#### 2. 页脚同步更新
**旧结构**：产品和服务（中台相关）、典型案例（零售/珠宝）、帮助与支持
**新结构**：服务、方案、产品与案例、关于我们

#### 3. 首页内容更新
- Hero区：AI驱动企业提效主题
- 新增 Work Agent 解决方案板块
- 新增 AI培训板块（Claude Code/OpenClaw）
- 行业深耕改为：医疗健康、智能制造、注意力培训
- CTA改为AI主题

#### 4. 移除内容
- 旧的中台相关描述（业务中台、数据中台、AI中台等）
- TempoSnap、TempoFlow 从导航移除（页面保留）
- 旧行业方案（零售、珠宝分销）

## 原始业务信息（2019年成立）

### 公司信息
- 深圳市速睿森视科技有限公司，成立于2019年
- 总部位于深圳市南山区高新区
- 核心价值观：创新、协作、诚信、卓越

### 业务范围
1. 软件研发与系统设计（后端、前端、小程序、移动端）
2. 企业数字化咨询服务
3. 产品和用户体验设计
4. AI应用开发（Agent、RAG、企业AI落地）

### 团队优势
1. 跨行业系统经验，打造合理产品逻辑
2. 以使用体验为导向，设计从实际场景出发
3. 快速响应高质量交付，项目服务灵活可靠
4. 高性价比解决方案，灵活合作方式

- 数智织机（服装制造ERP）
- 青少年认知专注力训练系统
